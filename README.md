# Edith AI Assistant (n8n Workflow)

AI-powered personal assistant built using n8n.

## Features
- Telegram chatbot
- Google Calendar scheduling
- Google Tasks management
- Expense tracker with Google Sheets
- Gmail automation
- OpenAI-powered AI agent
- Memory support

## Tech Stack
- n8n
- OpenAI GPT
- Telegram Bot API
- Google Calendar API
- Google Tasks API
- Google Sheets API
- Gmail API

## Workflow Overview
The assistant receives user messages from Telegram and intelligently routes tasks to the correct automation tool.

## Use Cases
- Schedule meetings
- Create tasks
- Track expenses
- Send emails
- Generate daily expense summaries

## Import Workflow
Download `Edith.json` and import it into n8n.
